## 디자인 (개정 이력 순)

1. 초기: 딥네이비+골드, 세리프 제목 → 폐기(2026-07-18)
2. 무채색 바탕 + 색은 신호로만, 포인트 블루 #2563eb, 골드는 카톡카드·인쇄물 전용으로 격하
3. 기준 기기: 태블릿(1024~1366) > 데스크톱(1920) > 모바일(390). 좌측 사이드바 + 모바일 햄버거. 검증 3해상도
4. 서체: 플랫폼 UI는 Pretendard 단일(현재 Variable). 세리프는 케어 아티클·카톡카드·인쇄물 전용
5. 밀도 원칙: 기준 실물 = 신한라이프 SOL FC의 일하는 화면. 조회 화면 문법(필터바+테이블+상세), 홈은 대시보드. 밀도는 실데이터로만 — 가짜 수치 금지. 메뉴 비대화(39개)는 반면교사, 메뉴 6개 유지
6. 화면 주인별 문법 분리: 설계사 화면 = 디자인+실무 효율 / 고객 화면(care·card·r) = 가독성+잡지급 퀄리티(경제지 편집 문법). 목표 수준 = 현존 업계 최고이자 앞으로도 최고
7. 케어 잡지: 채널 표지 아이덴티티(주간 네이비/월간 버건디), 섹션 컬러(시사 레드·경제 블루·교양 바이올렛·보험 그린), 모바일 스와이프 리더, 서재(표지 그리드)
8. 외부 시안(GPT 5.6) 수용 방식: 시안이 오면 디자인 언어만 추출해 CLAUDE.md 디자인 조항을 개정하고 전 화면에 일관 적용한다. 시안 코드를 그대로 붙이지 않는다
9. 현재 개선 방향 유지: 로고 단색, 타이포 낙차, 업무 도구 골격(헤더바·브레드크럼·검색창), 스탯 카드형 준비중 화면
