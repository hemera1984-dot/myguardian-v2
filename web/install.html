마이가디언 홈 바로가기 설치

마이가디언

어느 기기에서든 앱처럼 씁니다

홈 화면에 바로가기를 설치하면 브라우저 주소 입력 없이 한 번에 열립니다. 새 계정이나 별도 데이터가 생기지 않습니다 — 같은 마이가디언을 앱 모양으로 여는 것입니다.

설치 가능 여부 확인 중

아이폰·아이패드

  1. 사파리로 이 페이지를 엽니다
  2. 하단 공유 버튼을 누릅니다
  3. "홈 화면에 추가"를 선택합니다

안드로이드 휴대폰·태블릿

  1. 크롬으로 이 페이지를 엽니다
  2. 위의 "이 기기에 바로 설치"를 누릅니다
  3. 안내 창에서 설치를 확인합니다

데스크톱·노트북

  1. 크롬 또는 엣지로 엽니다
  2. "이 기기에 바로 설치"를 누르거나
  3. 주소창 오른쪽의 설치 아이콘을 누릅니다

설치가 안 되는 브라우저에서는 "앱 주소 복사"로 주소를 옮겨 크롬·사파리에서 다시 열어 주세요. 문의는 총관리자(안창민 FC)에게.
