발표 도구

브리핑

한 자료를 발표자 화면과 청중 화면 두 시점으로 띄웁니다. 페이지 넘김·마우스포인터·영상이 실시간으로 따라가고, 스크립트는 발표자 화면에만 보입니다.

모드 선택

기기 구성만 다를 뿐 화면과 조작은 같습니다.

내 라이브러리

탑재한 발표 자료 목차입니다. 제목을 누르면 바로 발표합니다. 이 브라우저에만 저장되며 저장소·외부로 전송되지 않습니다.

자료 선택

공개 자료

data/brief/에 발행된 자료입니다.

내 컴퓨터에서 열기

브리핑 문서(JSON)·PDF·HTML·이미지(여러 장)를 열어 발표합니다. 스크립트 파일(.txt/.md/.pdf/.html)을 함께 올리면 페이지별 발표자 스크립트가 됩니다 — 텍스트는 "---" 구분, PDF는 페이지가 구간, HTML은 수평선(hr) 구분. 파일은 이 브라우저 안에서만 처리되며 어디에도 전송되지 않습니다.

사용 순서: 자료를 열면 발표자 화면이 뜹니다 → "청중 화면 열기"로 새 창을 띄워 빔프로젝터·확장 모니터(상담은 고객 쪽 창)로 옮기고 F 키로 전체화면 → 발표자 화면에서 페이지를 넘기면 청중 창이 따라옵니다.
